                            function checkString(string){
                                pString = [];
                                for(index = string.length - 1; index >=0; index--){
                                    pString += string[index];
                                } 
                                return pString;
                            }

Code Structure
getString()

This function gets the inputted value from the user and changes it completely to lower case and replaces all other symbols with a blank space. It then calls checkString and if the returned string equals the original string, the boolean condition becomes True. Else, it becomes False. It then calls displayResult().

checkString()

This function runs a for loop from the length of the string - 1 to 0. It then adds these values to an empty list pString and returns it to the getString() function.

displayResult()

This function checks if the boolean condition is True or False. If it is True, it displays a result stating the inputted string is a palindrome and displays the string itself. Else, it displays that it is not a palindrome and displays the string. No matter what, the invisible class is removed from the HTML with the identifier "alert".
